== Description ==
[Documentation](https://compojoom.com/documentation/cforms-wordpress) | [Homepage](https://compojoom.com) | [Support](https://wordpress.org/support/plugin/cforms-plugin)

Tested with WordPress 4.8

Creating a form has never been faster! cForms is an the ultimate free drag & drop form builder plugin for WordPress. You can easily create beautiful and responsive forms with different fields, captchas, validations, payments and a lot more.

You can set the size for every form field and element you add and that for every device size individually. There are tons of different fields to choose for your form.

From a simple contact form over a complex order or booking form with cForms you can easily create them in no time!

= FEATURES: =

* Unlimited Forms
* Multiple pages
* Full Drag & Drop support
* Choose from different form designs
* Different Progress bars and headings

== Installation ==

Manual installation:

1. Upload `cforms-plugin` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

You will find the 'cForms' menu in your WordPress admin panel.

== Frequently Asked Questions ==

= What does cForms do? =
cForms Plugin allows you to build awesome forms for your WordPress site in no time. For example you can receive contact emails, create ordering forms or whatever you come up with.

= Where does cForms store the data? =

We store every form entry securely in the database, you can review, export or delete them in the cForms Submission View in your WordPress admin panel.
